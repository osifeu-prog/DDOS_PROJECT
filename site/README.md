# אתר משקיעים – SLH DDOS × BOT_FACTORY

תיקייה זו מכילה אתר סטטי למשקיעים, מוכן ל-GitHub Pages:

- `index.html` – דף נחיתה.
- `style.css` – עיצוב כהה ומודרני.

כדי להפעיל GitHub Pages:

1. ודא שהתיקייה `site/` נמצאת ב-branch (למשל `main`).
2. עבור ל-Settings → Pages והגדר:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /site
3. קבל URL ושלח למשקיעים.
