DDOS – Digital Democratic Operating System

I. עקרונות יסוד וחדשנות מרכזית

המערכת מתוכננת סביב עקרונות של **שקיפות מלאה**, **אי-שינוי** (באמצעות בלוקצ'יין) ו**אנונימיות מלאה** (באמצעות Zero-Knowledge Proofs - ZKP). ה-ZKP מבטיח שכל אזרח בעל זכות הצבעה יצביע רק פעם אחת, מבלי לחשוף את זהותו בפני המדינה או המערכת.

החדשנות העיקרית היא יצירת **"דמוקרטיה נזילה" (Liquid Democracy)** המשלבת דמוקרטיה ישירה (ברמה המקומית והארצית) עם אחריות מנהלית אוטומטית.

II. ארכיטקטורת המערכת (4 שכבות) [Image of a 4-layer system architecture diagram]

הארכיטקטורה מבוססת על גישת מיקרוסרביסים המחולקת לארבע שכבות מרכזיות:

  • שכבת הנתונים והתשתית: כוללת **בלוקצ'יין עצמאי (Permissioned Consortium)** המשתמש באלגוריתם Proof-of-Stake (PoS) לסילומיות גבוהה. הבלוקצ'יין משמש לרישום הצבעות, זהויות (ZKP) וחוקי המערכת (חוזים חכמים).
  • שכבת לוגיקה עסקית ("המוח" של המערכת): מנוע ניתוב נושאים, מנוע אוטומציה של החלטות וליבת ה-DDOS המטפלת בתהליכי הצבעה ואישור מנהלי.
  • שכבת שירותים (APIs): ממשקי RESTful API סטנדרטיים המאפשרים לכל גורם שלישי (כמו BOT_FACTORY) להתחבר לליבת ה-DDOS באופן מאובטח.
  • שכבת ממשקי משתמש: אפליקציות, אתרים ובוטים (כמו BOT_FACTORY) שמהווים את נקודת המגע של האזרח עם המערכת.

NDFS – National Digital Financial System

ה-NDFS היא שכבת הבלוקצ'יין הפיננסית. זו מערכת מודולרית המיועדת להקצאת תקציבים ושליטה ממוקדת בכספים ציבוריים באמצעות מטבע דיגיטלי.

ניהול פיננסי חכם וממוקד

  • ארנק ממוקד (Conditional Funds): ניתן להגביל חלק מכספי הסיוע (כגון קצבאות) לשימוש רק עבור שירותים חיוניים ספציפיים (כגון בתי מרקחת מאושרים או שירותי בריאות ציבוריים). זה מבטיח שהכסף מגיע ליעדו האמיתי ומקטין הונאות.
  • סיוע ליוקר המחיה: במקום סבסוד עקיף ליצרנים, המדינה יכולה להצביע (דרך מנגנון ה-DDOS) על **סבסוד ישיר וממוקד לצרכן**. הסבסוד מוזרם מיידית כהנחה אוטומטית בעת תשלום דרך ארנק ה-NDFS בבית העסק.

אומדן וכימות הבנקאות החדשה

ה-NDFS מהווה את התשתית למודל בנקאות וניהול מוניטרי חדש, המבוסס על נתונים בזמן אמת:

  • שקיפות כלכלית רגעית (Real-Time GDP/GNP): כל עסקה במטבע הדיגיטלי (אנונימית ברמת המשתמש, אך מקוטלגת וממוינת לפי סקטור כלכלי) נרשמת בבלוקצ'יין. זה מספק לבנק המרכזי ולמשרד האוצר **מדדי תמ"ג, תל"ג וצריכה בזמן אמת** ובגרנולריות חסרת תקדים.

BOT_FACTORY – שער המשקיעים וקהילת SLH

BOT_FACTORY הוא שירות FastAPI + Telegram Bot שרץ על Railway ומחובר לרשת **BNB Smart Chain**. הוא מהווה את ממשק המשתמש הראשוני והעיקרי לניהול השקעות ואינטראקציה עם האקו־סיסטם של DDOS.

תכונות עיקריות:

  • קישור כתובת BNB / SLH לפרופיל טלגרם.
  • ניהול יתרות Off-Chain (DB) מול קריאות On-Chain לכתובת הקהילה.
  • תפריט משקיעים מלא עם פקודות כמו /wallet, /balance, /history.
  • כלי ניהול משקיעים (admin) מוגנים, כגון /admin_credit.

דף זה נועד לשמש כשער ראשוני ברור. דפי התיעוד המלאים (Whitepaper / Investor Docs) יכולים לשבת בריפו זה או בדפי GitHub Pages נפרדים.
